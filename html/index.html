Card game

자바 스크립트를 이용하여 만든 카드 맞추기 게임입니다. 클릭 했을 때 같은 그림이 나오면 카드가 정지해 있고 다른 그림이 나오면 카드가 다시 뒤집어 지는 게임입니다.

Typing Game

자바 스크립트를 이용하여 만든 타이핑 게임입니다. 빨간 글씨로 표시하였고 맞은 글씨는 초록색으로 표시하였습니다. 남은 시간, 틀린 개수, 타이핑 속도를 표시하였습니다.

Music Player

자바 스크립트를 이용하여 만든 뮤직 플레이어입니다. 원하는 지점으로 이동이 가능하며 6개의 곡 목록이 있고 이전 곡, 다음 곡 버튼 플레이 버튼을 구현하였습니다.

Drawing Game

자바 스크립트를 이용하여 만든 드로잉 게임입니다. 5 색상의 브러쉬와 지우개를 이용하여 보드에 그림을 그릴 수 있고 도형 툴을 선택하여 원, 사각형, 삼각형을 그릴 수 있습니다.
